
## Description
This project is a basic version of PayTM, a payment application. It includes a backend built with Express and MongoDB, and a frontend developed using React and Tailwind CSS.

---

- [Description](#description)

---

## Backend
- **Express**: HTTP server framework.
- **MongoDB**: Database for storing user and transaction data.
- **Zod**: Input validation library.

---

## Frontend
- **React**: Frontend framework for building the user interface.
- **Tailwind CSS**: Styling framework for responsive and modern design.

---

##  Features
- User authentication (signin/signup).
- Secure payment processing.
- User friendly-interface.
